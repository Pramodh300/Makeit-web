Java

Java is a high-level programming language developed by Sun Microsystems. It was originally designed for developing programs for set-top boxes and handheld devices, but later became a popular choice for creating web applications.

// Get user input
Scanner scanner = new Scanner(System.in);
System.out.print("Enter your name: ");
String name = scanner.nextLine();
// Print a greeting
System.out.println("Hello, " + name + "! Welcome to the world of Java.");

This Java program demonstrates basic user input handling and output using the `Scanner` class. It first creates a `Scanner` object to read input from the user. Then, it prompts the user to enter their name and stores the input as a string. After receiving the input, the program prints a personalized greeting message using the entered name. This simple script showcases fundamental Java concepts such as standard input handling, string concatenation, and console output.
